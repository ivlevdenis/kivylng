### Instalation 

#### On Linux:

```bash
cd ~/.config/sublime-text-3/Packages/
git clone https://github.com/ivlevdenis/kivylng.git
```

#### On Windows:

```bash
cd %APPDATA%/Sublime Text 3/Packages/
git clone https://github.com/ivlevdenis/kivylng.git
```
#### On Mac:

```bash
cd ~/Library/Application\ Support/Sublime\ Text\ 3/Packages/
git clone https://github.com/ivlevdenis/kivylng.git
```
